Hello! Welcome to Autocode

This is a sample HTML file. Files in the www/ directory will automatically be converted into static webpages and resources.

Indexes

index.html / index.htm are special. They refer to the root directory and will be available at [example.com/service]/ as well as [example.com/service]/index.html.

Note: [example.com/service] means your service URL, usually something like username.api.stdlib.com/service@env, though you can define custom hostnames in your dashboard.

404 Error Pages (Not Found)

Name a file 404.html / 404.htm to create a 404 (not found) handler for a specific directory. It will automatically apply to any directory above it unless there's a higher priority 404 page found. This is similar to the behavior of functions/__notfound__.js files and will compete with them for priority.

Combining Endpoints and Static Files

You can use serverless functions (endpoints) and static files in the same project by having both a functions/ and a www/ directory. Projects can not have two files with the same route, meaning you can't have both functions/__main__.js and www/index.html in the same project.
